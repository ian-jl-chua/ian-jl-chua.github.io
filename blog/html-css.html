What are the best practices associated with using classes vs. ids?

The general rule would be to use class to style elements in a HTML document where ID is targeting a specific element of that HTML document.

Class

  • Is not unique and can be used to style multiple element on the document.

ID

  • IS unique and can only be used to style one element on the document.
