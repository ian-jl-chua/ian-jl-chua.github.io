What are the best practices associated with using classes vs. ids?

The general rule would be to use class to style elements in a HTML document where ID is targeting a specific element of that HTML document. If both class and id are used in an element, the id rule is applied if there are overlap in instructions.

For example if there is a 'font-weight: bold' in the class and a 'font-weight: lighter' in the id, and both of these are applied to an element. The 'font-weight: lighter' would be applied to the element.

NOTE: Class and ID selectors are case sensitive

Class

  • Is not unique and can be used to style multiple element on the document.
  • An element can also have more than one class selector.

ID

  • IS unique and can only be used to style one element on the document.
  • useful in conjunction with JavaScript
