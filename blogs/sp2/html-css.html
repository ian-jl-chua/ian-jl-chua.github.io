What are the best practices associated with using classes vs. ids?

The general rule would be to use class to style elements in a HTML document where ID is targeting a specific element of that HTML document. If both class and ID are used in an element, the ID rule is applied if there are overlap in instructions.

For example if there is a 'font-weight: bold' in the class and a 'font-weight: lighter' in the ID


and both of these are applied to an element.

The 'font-weight: lighter' would be applied to the element.

The way to look can be like a school. The "Class" are like class rooms in a school, and the "ID" are like the students. You can have many "Class"rooms but individual student "ID"s unique to each student.

NOTE: Class and ID selectors are case sensitive

Class

Is not unique and can be used to style multiple element on the document.

An element can also have more than one class selector. Here we can see two class selectors (one and two) in one element

IS unique and can only be used to style one element on the document.
