Explain control flow and loops using an example process from everyday life, for example, 'waking up' or 'brushing your teeth' (but not those).

By default JavaScript code is executed from top to bottom, line by line. This can be changed however with control flow. Control flow allows the programme to make decision about what code is executed, and when.

So it is like doing exercises, there are conditions. You can do different exercises to target different muscle groups in your body. These can be represented in the form of IF statements. Like if your legs are still not sore enough, do a few more squats. Having this done over and over again is like a loop, so keep doing squats until your legs are sore.

So together, the loop and conditionals are collectively known as control flow.

Describe what the DOM is and an example of how you might interact with it.

The DOM is a programing interface for web documents.It is a representation of a page so that programs can change the structure, style and content.

The DOM is a structure that web browsers use to make sense of the different things going on in the HTML, CSS and JS documents.

Inside the DOM everything are nodes. The elements, the attributes, comments, text, these are all nodes inside the DOM. So comparing this to the human anatomy, it is like studying the human anatomy and how each part of the body functions.

If you know the DOM, then you can use JS to change parts of the human structure, this is like interacting with it. So lets say you want to strengthen grip strength, you would need to grip related exercises by understanding the hands and forearm in the DOM.
